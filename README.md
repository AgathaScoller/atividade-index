## Sobre o Projeto

O **GamerStore** é um projeto desenvolvido durante o curso **Full-Stack Programming do SENAI**, com o objetivo de criar uma **loja virtual simples** utilizando **HTML, CSS, JavaScript e Bootstrap**.

O projeto simula um site de e-commerce voltado para produtos gamers, com funcionalidades como:

- Navegação por seções (Home, Destaques, Novidades, Login)  
- Carrossel de banners responsivo  
- Cards de produtos com botão de compra  
- Galeria de imagens  
- Formulário de login com validação básica via JavaScript  

> Este projeto é voltado para **prática acadêmica**, permitindo aplicar conceitos de front-end e manipulação de DOM em JavaScript.

---

## Funcionalidades

- **Carrossel de banners** totalmente responsivo  
- **Produtos em destaque** com botão de compra que exibe alerta  
- **Novidades** e seções informativas  
- **Galeria de imagens** usando grid do Bootstrap  
- **Login funcional** (validação simples)  
- Layout totalmente responsivo para dispositivos móveis
